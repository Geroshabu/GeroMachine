Member Function Documentation

void GeroMachine.StateMachine.InputTrigger ( Trigger  trigger)
inline

このステートマシンにトリガーを入力する

ステートマシンにトリガーを入力すると, このステートマシンが持っている状態遷移表に従って遷移が発生し, 現在の状態が遷移後の状態に更新される. トリガーに対する遷移が定義されていない場合は, 何もしない.

Parameters
trigger入力するトリガー
Exceptions
ArgumentNullException引数trigger がnull.
InvalidOperationException現在の状態が状態遷移表に存在しない. 状態遷移表の設定ミスなどによる.
